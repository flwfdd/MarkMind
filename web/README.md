# MarkMind - 个人知识管理与AI助手

## 功能特性

### 🏠 主页 - 知识图谱 + AI对话
- **交互式知识图谱可视化**
  - 支持4种知识图谱切换（前端开发、后端技术、AI/机器学习、产品设计）
  - 节点拖拽到聊天框自动生成相关问题
  - 实时搜索过滤，高亮匹配节点，变暗未匹配节点
  - 鼠标滚轮缩放（0.5x - 3x）
  - 鼠标拖拽平移视图
  - 缩放控制按钮（+/-/重置）和缩放比例显示
  
- **AI智能对话**
  - 流畅的消息交互界面
  - AI打字动画效果
  - 支持从知识图谱拖拽节点生成问题
  - 消息时间戳显示

### 📚 文档库
- **分类导航系统**
  - 5个文档分类（前端开发、后端技术、系统设计、数据结构与算法、产品设计）
  - 实时显示每个分类的文档数量
  - 活跃状态高亮显示
  
- **文档管理**
  - 文档列表展示（标题、更新时间、文件大小）
  - 文档上传功能
  - 响应式卡片布局
  - 悬停交互效果

### 🎓 学习助手
- **复习报告**：展示最近的学习回顾记录
- **吃灰收藏**：提醒长时间未阅读的收藏内容
- **待办事项**：管理学习任务
- **每日灵感**：随机展示激励性内容

## 技术栈

- **React 18** - 前端框架（Hooks: useState, useRef, useEffect）
- **Tailwind CSS** - 样式框架
- **Vite** - 构建工具
- **Lucide React** - 图标库
- **组件化架构** - 可复用组件库和主题系统

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产构建

```bash
npm run preview
```

## 核心功能亮点

### 知识图谱交互
- **多图谱切换**：前端、后端、AI/ML、设计四种主题
- **智能搜索**：输入关键词实时过滤节点
- **视图控制**：
  - 鼠标滚轮：缩放视图
  - 鼠标拖拽：平移画布
  - 控制按钮：精确缩放和重置
- **拖拽集成**：直接拖节点到聊天框生成问题

### 共享组件库
- `Card` - 统一的卡片容器
- `Button` - 主/次按钮变体
- `Badge` - 带计数的标签
- `IconBox` - 图标容器
- `ListItem` - 列表项卡片
- `TypingIndicator` - AI打字动画
